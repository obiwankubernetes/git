Creating and Pushing a New Repo
1. create repo on github 
1. check if git is running in command line `git`  
1. initialize git `git init`  
1. adding untracked files `git add .`  
1. check status of staged files `git status`  
1. commit it with a message (name) `git commit -m "name"`  
1. conncecting local to github cloud (two steps from github):
    - `git remote add origin <link>`
    - `git push -u origin master`



Modify a Repo  
1. adding untracked files `git add .`  
1. check status of staged files `git status`  
1. commit it with a message (name) `git commit -m "name"` 
1. push modifications `git push`
